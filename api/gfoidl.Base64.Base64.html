DetectEncoding(ReadOnlySpan<Byte>, Boolean)

Detects whether encoded is base64 or base64Url.

Declaration
public static EncodingType DetectEncoding(ReadOnlySpan<byte> encoded, bool fast = false)
Parameters
Type Name Description
System.ReadOnlySpan<System.Byte> encoded

The base64 encoded data.
System.Boolean fast

When false (default) encoded is scanned one time for base64 chars and a second time for base64Url chars. So if there is a mix of them, Unknown will be returned.

When true encoded is scanned only once and for base64Url chars. So if there is a mix of base64 and base64Url, the result will be Base64Url, and may throw a System.FormatException on decoding.

Returns
Type Description
EncodingType

base64 or base64Url
Remarks

It is an O(n) scan / detection of the encoding type, and input is not validated for conforming the base64 standard. Thus there is no 'Invalid' encoding type.
